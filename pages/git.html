Pourquoi git ?

Pour notre projet Slime GO nous avons utilisé gitlab qui nous a permit en partie de se répartir le travaille de s'organiser et de faire des sauvegardes et différentes versions du projet. Nous avons appris a utilisé git tout au long de notre formation pour les projets en groupes ou à plusieurs et c'est un outil intéressant puisqu'il est très utile. Depuis j'utilise git pour la plus part de mes projets.
